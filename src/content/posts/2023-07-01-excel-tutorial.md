## 常用函数

### 统计函数

```excel
=SUM(A1:A10)      # 求和
=AVERAGE(A1:A10)   # 平均值
=COUNT(A1:A10)     # 计数
=MAX(A1:A10)       # 最大值
=MIN(A1:A10)       # 最小值
```

### 逻辑函数

```excel
=IF(A1>100,"达标","未达标")
=AND(A1>100,B1>100)
=OR(A1>100,B1>100)
```

### 查找函数

```excel
=VLOOKUP(查找值,表格范围,列号,匹配类型)
=INDEX(范围,MATCH(查找值,查找范围,匹配类型))
```
